# Next.js Blog Application

This is a full-stack blog application built for assignment submission.

## Tech Stack

- Next.js
- Prisma ORM
- PostgreSQL (Supabase)
- NextAuth.js for authentication

## Features

- Signup/Login
- Create Blog Post (only logged-in users)
- Public Blog List
- Blog Detail View
- Blog Deletion (only for authors)
